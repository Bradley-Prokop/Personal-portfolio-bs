Hangman

This is a simple hangman game built with html, css, and jquery. You have 7 lives to guess the phrase or you lose. You can either guess a letter or the entire phrase at a time, each counts as one life lost if incorrect. It will keep track of the best score for each phrase and will update it in local storage. You can either click submit or the enter key to submit a guess.

BMI calculator

This is a simple BMI (body mass index) calculator built with html, css, and Javascript. This is a pretty simple project that allows the user to calculate their BMI. It allows them to choose between the metric and imperial systems.
